How to use this calculator

GPA is a different grading system from WAM, and is usually calculated on 4.0 or 7.0 scale.

The following table will demonstrate the value of each grade on both scales:

Grade Value (4.0) Value (7.0)
HD High Distinction 4.0 7.0
D Distinction 3.0 6.0
Cr Credit 2.0 5.0
P Pass 1.0 4.0
F Fail 0 0
